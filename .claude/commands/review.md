현재 브랜치의 변경사항을 코드 리뷰한다.

`git diff main...HEAD` 또는 staged 변경사항을 대상으로 아래 순서로 검토:

1. **버그** — 런타임 오류, 엣지 케이스 누락, 잘못된 조건문
2. **보안** — XSS, SQL injection, 민감 정보 노출, 인증 누락
3. **타입** — TypeScript 타입 누락, `any` 남용, 타입 불일치
4. **성능** — 불필요한 리렌더, 무한 루프 가능성, 큰 번들
5. **간결성** — 중복 코드, 불필요한 추상화, 죽은 코드

결과는 **심각도(high/medium/low)** 와 **파일:라인** 을 포함해 bullet으로 출력.
문제 없으면 "LGTM" 과 함께 잘된 점 한 줄 언급.
